Building a modern
single page web app

I chose to recreate the Specialized Website with a more narrow focus on mountain bikes due to the fact that I love to mountain biking and ride a Specialized StumpJumper so there was a personal connection to the brand. I also feel their website accurately portrays a modern E-Commerce Website of a global company.

When deciding on how to build this site I wanted to take advantage of the many benefits a component based JavaScript Library/Framework makes available to developers and users. I have become quite confident building with React and personally feel that it makes my work flow more efficient and enables me produce websites with higher quality and predictability.

The Tech Behind
The Website

Throughout my Web Development career I have realized just how powerful React can be in creating a performant Website. From its faster rendering through the use of the virtual DOM, enhancing development productivity and bettering future maintenance, React is my tool of choice.

Along with a powerful Javascript library, other Web essentials make a build more stable and robust. I used WebPack to bundle and minify my code and assets. In addition, I find that utilizing SASS makes styling my website easier and more intuitive than traditional CSS. Because React runs on Javascript in makes sense to couple it with Node and make use of NPM to manage all my other libraries and dependencies. And Finally, Git and Github, to ensure my versioning and documentation is readily accessible.
